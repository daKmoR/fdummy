Usage is always pretty much the same:

  1. load the needed files (you just need to define the actual file/package you actually need, dependences will be resolved automatically)
    • in fluid*: ]]>
    • with ts**: plugin.tx_mootoolsessentials.settings.load.files.10 = More-Behaviors/Behavior.OverText
  2. use the behavior with HTML5

*: you will need to define the namespace "m" with {}

**: the "name" like in this example 10 is completely irrelevant as long as it's unique. We sugguested you use the name of the behavior e.g. .files.OverText = More-Behaviors/Behavior.OverText
